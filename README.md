# vibranite Minecraft Mod

## About 
A Minecraft mod build for Fabric. It adds a new type of ore, as well as new weapons, tools, and armor!

## How to install for Minecraft Java:
Currently, the mod has not yet been verified on Modrinth, so it is not yet available on the Modrinth launcher.
To install and try the mod yourself, 

1. Make sure you have Java 21+ installed on your computer, as well as Fabric 26.2 installed.
2. Use the google drive link to download the .jar file
3. go to modrinth.com and download the Fabric API mod for 26.2
4. navigate to your mods folder (can be accessed from the installations tab on the Minecraft launcher)
5. paste the Fabric API and Vibranite Mod into your mods folder
6. Run the Fabric 26.2 Installation!
